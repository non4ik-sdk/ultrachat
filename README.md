# UltraChat
Simple Ultrakill styled Twitch chat for stream
1. Download as .zip or other.
2. Add .html path to obs browser widget.
3. Open `config.js`
4. Set channel name (not full url).
5. Enable/Disable 7TV.
6. Setup proxy or use my [CF Worker](https://github.com/non4ik-sdk/CacheProxy).
7. Reload widget.

Why do you need a proxy?
- **If you're in Russia**, you'll need to use a **proxy to load 7tv emotes**. If you don't need this, just leave this line blank.

Is authorization required?
- No
  
Font?

- Support Cyrillic and Latin
